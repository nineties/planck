=================
Bytecode Encoding
=================

File format
===========

+--------------+--------------------------------+
| field length | content                        |
+==============+================================+
| 4 bytes      | File magic                     |
|              |                                |
|              | * object file: "PLKO"          |
|              | * executable file: "PLKE"      |
+--------------+--------------------------------+
| 4 bytes      | Data size (N)                  |
+--------------+--------------------------------+
| N bytes      | Sections                       |
+--------------+--------------------------------+

Section Format
==============

+--------------+--------------------------------+
| field length | content                        |
+==============+================================+
| 4 bytes      | Section type                   |
|              | 4 ascii characters like "Code" |
+--------------+--------------------------------+
| 4 bytes      | Section size (N)               |
+--------------+--------------------------------+
| N bytes      | Section dependent data         |
+--------------+--------------------------------+
| (N - 3)/4*4  | Padding to fit 4-byte boundary |
| bytes        |                                |
+--------------+--------------------------------+

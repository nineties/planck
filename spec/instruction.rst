===============
Instruction Set
===============

Non-Branch Instructions
=======================

::

   +---------------+----------+-----------------+------------------------------+
   | name          | 1st byte | following bytes |                              |
   +===============+==========+=================+==============================+
   | nop           | 00000000 |                 | do nothing                   |
   +---------------+----------+-----------------+------------------------------+

Branch Instructions
===================

::

   +---------------+----------+-----------------+------------------------------+
   | name          | 1st byte | following bytes |                              |
   +===============+==========+=================+==============================+
   | return        | 10000000 |                 | return from function         |
   +---------------+----------+-----------------+------------------------------+

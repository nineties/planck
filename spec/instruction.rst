===============
Instruction Set
===============

Non-Branch Instructions
=======================

::

   +---------------+----------+-----------------+------------------------------+
   | name          | 1st byte | following bytes |                              |
   +===============+==========+=================+==============================+
   | nop           | 00000000 |                 | do nothing                   |
   +---------------+----------+-----------------+------------------------------+
   | move          | 00000001 | opd0,opd1       | opd0 = opd1                  |
   +---------------+----------+-----------------+------------------------------+

Branch Instructions
===================

::

   +---------------+----------+-----------------+------------------------------+
   | name          | 1st byte | following bytes |                              |
   +===============+==========+=================+==============================+
   | goto          | 10000000 | uint(i)         | do nothing, proceed to the   |
   |               |          |                 | basic block i.               |
   +---------------+----------+-----------------+------------------------------+
   | return        | 10000001 |                 | return from function         |
   +---------------+----------+-----------------+------------------------------+

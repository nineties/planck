Instruction Types
=================

+---------------------------+---------------------------+
| name                      | notation                  |
+===========================+===========================+
| builtin operations        | y = op(x1, ..., xn)       |
+---------------------------+---------------------------+
| function call             | y = f(x1, ..., xn)        |
+---------------------------+---------------------------+
| phi function              | y = phi(x1, ..., xn)      |
+---------------------------+---------------------------+
| returning from a function | return x                  |
+---------------------------+---------------------------+
| unconditional branch      | goto label                |
+---------------------------+---------------------------+
| conditional branch        | branch cond labl          |
+---------------------------+---------------------------+

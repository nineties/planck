Language Syntax
===============

::
   id    ::= [A-Za-z_][A-Za-z0-9_]*
   char  ::= ' [^'\] '
           | '\ [0abtnvfr'\] '
   int   ::= (+|-)? [1-9][0-9]*
           | (+|-)? "0" [0-7]*
           | (+|-)? "0x" [0-9A-Fa-f]+
           | (+|-)? "0b" [01]+
   float ::= (+|-)? [0-9]+ . [0-9]* ((e|E) (+|-)? [0-9]+)?
   str   ::= " ([^"\]|\[0abtnvfr"\])+ "
